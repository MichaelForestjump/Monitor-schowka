Clipboard monitor. My first useful program written on February 27, 2018 in C # WPF.

Functions:
- reading of the Windows clipboard (pasting into the program window)
- you can modify the text and copy to the clipboard
- you can start the narrator who will read the displayed text
- you can write your own text to let the narrator read it
- clipboard monitoring (refreshes and shows current content)
- the application is always displayed on top
- the starting position of the application is the upper right corner of the screen

Monitor schowka. Mój pierwszy użyteczny program napisany 27 lutego 2018 w C# WPF.

Funkcje:
- odczyt schowka Windows (wklejanie do okna programu)
- możesz zmodyfikować tekst i skopiować do schowka
- możesz uruchomić narratora który przeczyta wyświetlany tekst
- możesz napisać własny tekst by narrator go odczytał
- monitorowanie schowka (odświeża i pokazuje aktualną zawartość)
- apikacja wyświetla się zawsze na wierzchu
- pozycja startowa aplikacji to prawy górny róg ekranu
